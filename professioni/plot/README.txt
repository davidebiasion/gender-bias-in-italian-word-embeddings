How to understand file names:

<embeddingtype>_<genderdirection>

<embeddingtype> : ft (FastText), ft_deb (FastText with grammatical gender debiasing), g (Gonen)

<genderdirection> : g_diff ("lui"-"lei"), g_pca_0 (first PCA component), g_pca_01 (weighted first and second PCA components), g_e (pseudo LDA)
